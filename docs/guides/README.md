## Contributing

To add a new guide:

1. Create a new .md file in this directory
2. Follow the guide template format:
   - Clear title and objective
   - Table of contents for long guides
   - Prerequisites section
   - Step-by-step instructions with examples
   - Code snippets and screenshots where helpful
   - Expected outcomes
   - Troubleshooting tips
   - Links to related resources
3. Update this README with a link and description
4. Add to appropriate category
5. Update Quick Navigation if applicable

### Guide Writing Best Practices

- **Be comprehensive** - Cover topic thoroughly
- **Use examples** - Show don't just tell
- **Include visuals** - Mermaid diagrams, code blocks
- **Link extensively** - Cross-reference other docs
- **Keep updated** - Maintain with SDK changes
- **Target audience** - Write for intended skill level
