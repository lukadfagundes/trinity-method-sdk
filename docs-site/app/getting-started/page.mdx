# Getting Started with Trinity Method SDK

Welcome to the Trinity Method SDK! This guide will help you get up and running with investigation-first development in minutes.

## What is Trinity Method?

Trinity Method is an **investigation-first development methodology** that helps you build better software through systematic investigations powered by AI agents. Instead of jumping straight to coding, you start with structured investigations that uncover insights, identify patterns, and guide your development.

## Quick Start

### Installation

```bash
npm install -g @trinity-method/sdk
```

Or with your project:

```bash
npm install @trinity-method/sdk
```

```bash
trinity init
```

### Your First Investigation

Create a security audit investigation:

```bash
trinity create investigation --type security-audit
```

Or use the interactive wizard:

```bash
trinity wizard
```

The wizard will:
1. Detect your project's framework and language
2. Suggest relevant investigation types
3. Generate a comprehensive investigation plan
4. Set up all necessary files and structure

## Core Concepts

### Investigations

Investigations are systematic explorations of your codebase to uncover insights:

- **Security Audits** - Find vulnerabilities and security issues
- **Performance Reviews** - Identify bottlenecks and optimization opportunities
- **Architecture Analysis** - Understand and document system architecture
- **Code Quality** - Assess maintainability and code health

### AI Agents

Trinity uses specialized agents for different tasks:

- **TAN** (Technical Analysis) - Deep technical investigation and analysis
- **ZEN** (Pattern Recognition) - Identifies patterns and best practices
- **INO** (Research) - Gathers context and external knowledge
- **JUNO** (Quality Audit) - Validates quality and completeness

### Sessions

Each investigation runs in a session that captures:
- Context and decisions made
- Findings and insights discovered
- Work completed and next steps
- Metrics and performance data

## Prerequisites

- Node.js 18+ or 20+
- npm or pnpm
- Git (recommended)
- Claude Code, Cursor, or GitHub Copilot (for AI agent integration)

## System Requirements

- **OS**: Windows, macOS, or Linux
- **Memory**: 4GB+ RAM recommended
- **Disk**: 100MB for SDK + space for investigations
